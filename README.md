# Excel Business Performance Dashboard

## Overview
End-to-end Excel analytics project using Kaggle sales data. The project includes data cleaning, KPI creation, pivot analysis, interactive dashboard and business insights.

## Key Features
- Cleaned and validated 1,400+ records
- Removed 3,500+ duplicate rows
- Built KPIs for Sales, Profit, Margin and Orders
- Created Pivot Tables with calculated fields
- Designed interactive dashboard with slicers
- Analyzed regional and category profitability

## Tools Used
- Microsoft Excel
- Pivot Tables
- Calculated Fields
- Conditional Formatting
- Slicers

## Business Insights
- NSW contributes majority of revenue (regional dependency risk)
- Office Supplies is the main profit driver
- Technology shows strong sales but lowest margin
- Furniture has higher margin but low volume
